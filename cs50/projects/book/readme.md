Text Processing
This Python script shows basic text file handling in Python using open(), read(), readlines(), and writelines() with the file alice.txt.

Features
Read Entire File
Returns all the text from alice.txt as one string.

Read Lines
Returns every line of alice.txt as a list item.

Extract Chapter 1
Copies the lines for Chapter 1 from alice.txt into a new file chapter1.txt.


Usage
Put alice.txt in the same folder as the script.
Run the script with Python:
python book.py


chapter1.txt will be created with Chapter 1 of the book.
