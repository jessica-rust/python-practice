# Web Scraping Projects

This folder contains Python web scraping scripts demonstrating different scraping approaches.

## GreenSource Requests Scraper
- Uses `requests` and `BeautifulSoup`
- Handles pagination and Cloudflare-protected email addresses
- Outputs a CSV file of green businesses

## Green Business Selenium Scraper
- Uses Selenium for JavaScript-rendered content
- Scrolls dynamically loaded pages and opens detail pages
- Extracts business information and exports to CSV
